# 📱 Screenshots do App

## Como Capturar Screenshots

### 📋 **Lista de Screenshots Necessárias:**

1. **login_screen.png** - Tela de login inicial
2. **task_list.png** - Lista principal de tarefas
3. **add_task.png** - Formulário de nova tarefa
4. **date_time_picker.png** - Seletores de data e hora
5. **location_map.png** - Mapa de seleção de localização
6. **overdue_tasks.png** - Tarefas vencidas destacadas

### 🎯 **Como Capturar:**

#### No Emulador Android:
1. Pressione `Ctrl + Shift + S` (ou use o botão de câmera no painel lateral)
2. Ou use o comando: `flutter screenshot`

#### No Dispositivo Físico:
- **Android**: Power + Volume Down
- **iOS**: Power + Volume Up

### 📐 **Recomendações:**

- **Resolução**: Preferencialmente 1080x1920 ou similar
- **Formato**: PNG para melhor qualidade
- **Orientação**: Portrait (vertical)
- **Conteúdo**: Inclua dados de exemplo realistas

### 🎨 **Cenários para Capturar:**

#### 1. **Login Screen** (`login_screen.png`)
- Tela de login limpa
- Mostrar os campos de usuário/email e senha

#### 2. **Task List** (`task_list.png`)
- Lista com 4-5 tarefas variadas
- Incluir tarefas concluídas e pendentes
- Mostrar diferentes tipos (com/sem localização, com/sem vencimento)

#### 3. **Add Task** (`add_task.png`)
- Formulário parcialmente preenchido
- Mostrar todos os campos disponíveis
- Incluir seção de data/hora e localização

#### 4. **Date Time Picker** (`date_time_picker.png`)
- Seletor de data aberto ou
- Tela mostrando data/hora selecionadas

#### 5. **Location Map** (`location_map.png`)
- Mapa com marcador posicionado
- Mostrar a interface do seletor de localização

#### 6. **Overdue Tasks** (`overdue_tasks.png`)
- Lista com tarefas vencidas destacadas
- Mostrar badges vermelhas "VENCIDA"
- Demonstrar o visual diferenciado

### 🔄 **Após Capturar:**

1. Renomeie os arquivos conforme a lista acima
2. Coloque todos na pasta `screenshots/`
3. Verifique se as imagens estão sendo referenciadas corretamente no README.md

### 💡 **Dicas Extras:**

- Use dados realistas nas tarefas (ex: "Reunião com cliente", "Comprar ingredientes para jantar")
- Inclua emojis nas descrições para tornar mais visual
- Varie as datas (hoje, amanhã, datas passadas para mostrar vencimentos)
- Teste diferentes localizações no mapa

### 🎬 **GIF Animado (Opcional)**

Para criar um GIF demonstrativo:
1. Use ferramentas como **LICEcap** ou **Gyazo GIF**
2. Grave uma sequência de: Login → Lista → Adicionar Tarefa → Salvar
3. Salve como `app_demo.gif`
4. Adicione no README principal
